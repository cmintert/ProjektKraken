commands.base_command module¶

Base Command Module.

Defines the abstract base class and result type for all commands in the application.

Classes:

CommandResult: Standardized result object for command execution. BaseCommand: Abstract base class implementing command pattern with undo/redo.

commands.longform_commands module¶

Longform Commands Module.

Provides command classes for manipulating longform document structure: - MoveLongformEntryCommand: Move an entry to a new position - PromoteLongformEntryCommand: Reduce depth and change parent - DemoteLongformEntryCommand: Increase depth and reparent to sibling - RemoveLongformEntryCommand: Remove from longform document

All commands support undo/redo operations and return CommandResult objects.

commands.wiki_commands module¶

Wiki Commands Module. Commands for processing WikiLinks and updating relations.

class commands.wiki_commands.ProcessWikiLinksCommand(source_id: str, text_content: str, field: str = 'description')[source]¶

Bases: BaseCommand

Command to process text content, extract WikiLinks, and create ‘mentions’ relations.

This command: - Parses WikiLinks from text content - Resolves names to entities (case-insensitive, including aliases) - Creates ‘mentions’ relations with metadata (snippet, offsets, field) - Skips ambiguous matches (multiple entities with same name/alias) - Deduplicates by (target_id, start_offset)

__init__(source_id: str, text_content: str, field: str = 'description') → None[source]¶

Initializes the command.

Parameters:

  • source_id – The ID of the source entity or event.

  • text_content – The text content to parse for WikiLinks.

  • field – The field name where the text is stored (default: “description”).

execute(db_service: DatabaseService) → CommandResult[source]¶

Executes the link processing.

Extracts WikiLinks, resolves them to entities or events, and creates ‘mentions’ relations with metadata including field, snippet, and offsets.

Result of the operation.
